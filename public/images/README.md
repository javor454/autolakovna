# Before/After Images

Add your own before/after photos here:

1. **before.jpg** - Photo of damaged/scratched car (BEFORE repair)
2. **after.jpg** - Photo of same car after perfect paint job (AFTER repair)

## Recommendations:
- Same car, same angle for both photos
- Similar lighting conditions
- High quality (at least 1920x1080)
- Show clear difference between damaged and repaired state
- JPG format, optimized for web (under 500KB each)

Once you add these files, the slider will automatically use them.
